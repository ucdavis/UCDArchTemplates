UCDArch Templates provides T4 templates for MVC2+ which suppliment Add View/Controller.  Works well with with Twitter Bootstrap and UcdArch.

UCDArch is a framework built on ASPMVC and can be found here: http://ucdarch.codeplex.com/

Biggest Differences from ASPMVC default templates

*All modification templates will use Html.AntiForgeryToken()

*The Edit/Create page will provide guidance on using a partial template

*Client validation will be enabled by default
